# Launcher State

![launcher-state](./launcher-state.drawio.svg)

| State    | Description                                                                             |
| -------- | --------------------------------------------------------------------------------------- |
| WAITING  | Autoware は起動していません。バージョンの変更やファイルの更新などを行うことができます。 |
| CHANGING | Autoware のバージョンを変更しています。                                                 |
| RUNNING  | Autoware が起動しています。起動中の詳細な状態は各種ステートを参照してください。         |
